Design Summary



Why this design?

Since I love cars, I decided to make a website that involves selling cars. My audience are any people interested in buying cars from all walks of life.

I chose to use a carousel on the home page in order to attract clearly give users info about the website. I maintained the same architecture across all pages but with pictures instead for consistency.

For my choice of color theme, I based it on the pictures I used to make the website.

Tools used
  • Html 5

    I used html 5 to structure the various web pages and their content.

  • CSS

    I used to css to style up the pages inorder to make them more attractive and stylish.

  • Javascript and Jquery

    I used this two combined to help me enable users add items to cart and calculate their final cost. I also used them to style the navbar when the user scrolls the page.

  • Bootstrap

    I used to bootstrap to help me style up the site with predefined css code.

  • Custom fonts

    I downloaded some custom fonts from google fonts website and used them in the application to improve the site's aesthetics.

Features of the site

  • Ability to view all products in the system

    The site makes it easy for users to view the different types of vehicles in the system. This was achieved by showing 3 vehicles on the home page then providing the users with a button to take them to the inventory page where they can see all cars and easily add them to cart.

  • Cart system

    The site has the ability to enable users to add different cars of different counts in their cart just like in supermarkets. This feature was achieved using jquery and javascript and also using the local storage on the browser to keep the data persistent even after page is reloaded.

    To add and item to cart, just hover on the image of the vehicle that caught your eye and you will see an Add to cart button. Click it and it will add that item to cart.

  • Ability to see order summary

    I added an order summary page to enables users to view a detailed report of the cost they would incur for transparency. The page also shows a summary of the items in the cart.

  • Ability to view all reviews

    The site gives users a chance to see all reviews made about the site inorder to help know the value of the business. The feature was achived by using the bootstrap carousel feature and a little bit of css to spice up the design.

  • Navbar feature

    For ease of use of the website, I added the navbar to help users quickly get to where they want to go within the site.

  • About us page

    For information purpose, the about us page was added to help users know more about the site.

  • Contact us page

    Though not fully done, this page is meant to help users send an email to me incase they have any inquires
